Resources Page of Paul Piwek

  • Version 1.0 of the CODA Corpus, a parallel corpus of annotated monologue and dialogue.
  • ALLIGATOR Open Source Theorem Prover for Dependent Type Systems version 1.0 released April 2006 (source code in both Sicstus and SWI Prolog available). The prover deals with inferences in propositional logic, predicate logic and higher order logics.
  • Mapping algorithm for Piwek, P. (2007). Meaning and Dialogue Coherence: a Proof-theoretic Investigation. Journal of Logic, Language and Information, 16(4):403-421. [final draft .pdf]. The algorithm - implemented in SWI Prolog - maps proof (search) trees to dialogue structures.
  • Mapping algorithm for Piwek, P. (2011). Dialogue Structure and Logical Expressivism. To appear in: Special Issue on of Synthese KRA on "Theories of Information Dynamics and Interaction and their Application to Dialogue". Guest editors: Laure Vieu and Emiliano Lorini
  • The CODA project is producing a number of M2D resources related to the automated generation of dialogue scripts from text, that is, Monologue-to-Dialogue (M2D) transformations.
